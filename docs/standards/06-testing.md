# 06 테스트

## 목적

기능, 통합, 회귀와 검색 품질을 검증할 기준을 정의합니다.

## 필수 산출물

- 단위·통합 테스트 범위
- 핵심 사용자 시나리오
- 검색 정확도와 성능 평가 기준

## 완료 기준

- 성공 조건이 측정 가능한 값으로 정의되어 있다.
- 실패와 경계 조건이 검증된다.
- 동일 명령으로 검증 결과를 재현할 수 있다.
