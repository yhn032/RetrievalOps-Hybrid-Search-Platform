# 05 코딩

## 목적

설계를 구현할 때 적용할 코드 구성, 변경 단위와 품질 기준을 정의합니다.

## 필수 산출물

- 코드 구조와 명명 기준
- 의존성 및 설정 관리 기준
- 검토 가능한 변경 단위

## 완료 기준

- 구현이 승인된 설계와 연결된다.
- 비밀정보가 코드와 추적 파일에 포함되지 않는다.
- 빌드와 정적 검사가 재현 가능하다.
