# 04 API

## 목적

외부와 내부 API의 계약, 오류 처리와 인증 경계를 정의합니다.

## 필수 산출물

- endpoint와 요청·응답 계약
- 오류 형식과 상태 코드
- 인증·인가 및 버전 관리 기준

## 완료 기준

- 정상·오류 흐름이 명세되어 있다.
- 데이터 계약과 책임 주체가 명확하다.
- 호환성 변경 정책이 정의되어 있다.
