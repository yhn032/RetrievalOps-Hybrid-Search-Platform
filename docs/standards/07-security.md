# 07 보안

## 목적

데이터, 인증정보, 의존성과 외부 연동에서 발생할 위험을 식별하고 통제합니다.

## 필수 산출물

- 위협과 신뢰 경계
- 비밀정보 및 민감 데이터 처리 기준
- 의존성 취약점 대응 기준

## 완료 기준

- 민감 데이터가 추적 대상과 분리되어 있다.
- 인증·인가 실패 흐름이 검증되어 있다.
- 알려진 위험과 수용 근거가 기록되어 있다.
